# Лабораторна робота 7: Continuous Integration

## Тестовий репозиторій для GitHub Actions

Цей репозиторій використовується для налаштування та тестування процесу Continuous Integration (CI) за допомогою GitHub Actions, згідно з вимогами лабораторної роботи 7.

CI налаштовано на автоматичний запуск тестів при кожному оновленні гілки main або при створенні Pull Request.
